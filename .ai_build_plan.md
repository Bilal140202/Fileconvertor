# Fileconvertor - AI Build Blueprint (Image Suite)

This repository contains a high-performance, browser-based image conversion and optimization suite.

## Goal
To provide a privacy-first, client-side tool for bulk image processing, format conversion, and compression using modern web technologies.

## Tech Stack
- **Frontend:** React 19 with Zustand for state management.
- **Engine:** Sharp (WASM) and @squoosh/lib for high-performance processing.
- **Workers:** Web Workers for non-blocking UI during heavy operations.
- **Packaging:** JSZip for bulk downloads.

## AI Guidelines
1. When adding new formats, ensure both WASM and fallback implementations are considered.
2. Maintain strict separation between the processing logic (\lib/\) and UI components (\components/\).
3. All new conversion modules must include unit tests in the \	ests/\ directory.
4. Optimize for large file handling by implementing chunked processing in Web Workers.
