## 🌟 Overview
**Fileconvertor** is a powerful, privacy-first image processing suite that runs entirely in your browser. By leveraging WebAssembly (Sharp WASM) and modern Web Workers, it provides server-grade image optimization and conversion without ever uploading your files to a server.

## 🚀 Key Features
- ⚡ **Lightning Fast:** Uses WebAssembly for near-native processing speeds.
- 🛡️ **Privacy Centric:** All conversions happen locally on your machine.
- 📦 **Bulk Processing:** Convert and optimize hundreds of images simultaneously.
- 🎨 **Multi-Format Support:** Convert between PNG, JPEG, WebP, AVIF, and more.
- 📉 **Advanced Compression:** Fine-tuned control over quality and file size using Squoosh libraries.

## 🛠️ Built With
- **React 19:** For a reactive and performant user interface.
- **Sharp (WASM):** High-performance image processing engine.
- **Zustand:** Lightweight state management.
- **Vitest:** Comprehensive testing suite for reliability.
